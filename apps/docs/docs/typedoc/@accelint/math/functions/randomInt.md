# Function: randomInt()

> **randomInt**(`min`, `max`): `number`

Generate a random integer within the given bounds.

## Parameters

### min

`number`

### max

`number`

## Returns

`number`

## Throws

Throws an error if min > max.

## Example

```ts
const value = randomInt(0, 10);
// value >= 0 && value <= 10;

const value = randomInt(10, 0);
// RangeError
```
