# Function: random()

> **random**(`min`, `max`): `number`

Generate a random number within the given bounds.

## Parameters

### min

`number`

### max

`number`

## Returns

`number`

## Throws

Throws an error if min > max.

## Example

```ts
const value = random(0, 10);
// value >= 0 && value <= 10;

const value = random(10, 0);
// RangeError
```
