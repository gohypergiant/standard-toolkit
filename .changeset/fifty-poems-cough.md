Add comprehensive color conversion utilities with full CSS spec support. Convert between deck.gl tuples (0-255), GLSL tuples (0-1), CSS RGBA (string/tuple/object), and hex strings. Includes all hex formats (3, 4, 6, 8 character), modern and legacy CSS RGBA syntax, type guards, and lossless round-trip conversions. Optimized for performance with 97% test coverage.
