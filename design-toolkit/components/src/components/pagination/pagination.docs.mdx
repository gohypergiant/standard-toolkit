# Pagination
The Pagination component provides a simple way to navigate multiple pages while leveraging existing Button components.

It is composed of the following components:

- **Pagination** - A wrapper component for containing all pagination buttons.
- **Pagination.Provider** - Context provider for pagination display state.
- **Pagination.Previous** - Button to navigate backwards in page count.
- **Pagination.Next** - Button to navigate forwards in page count.
- **Pagination.NumberContainer - Wrapper for page buttons, displaying a maximum range of 5 pages.


## Example usage
```tsx
<Pagination.Provider>
  <Pagination>
    <Pagination.Previous />
    <Pagination.NumberContainer />
    <Pagination.Next />
  </Pagination>
</Pagination.Provider>
```
