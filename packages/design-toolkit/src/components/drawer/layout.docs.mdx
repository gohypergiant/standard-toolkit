# DrawerLayout
A CSS Grid-based layout system for positioning panels on any side of the main content area. Shared layout primitive used by [Drawer](?path=/docs/components-drawer--playground) and [Sidenav](?path=/docs/components-sidenav--playground).

<Canvas of={DrawerLayoutStories.Default} />

## Example Usage

```tsx
<DrawerLayout extend="left right" push="left">
  <DrawerLayoutMain>
    {/* Main content area */}
  </DrawerLayoutMain>

  {/* Drawer, Sidenav, or other panel components */}
</DrawerLayout>
```

## Props

### DrawerLayout

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `extend` | `'top bottom' \| 'left right' \| 'top' \| 'bottom' \| 'left' \| 'right'` | `'left right'` | Which panels span full container dimensions |
| `push` | `'top' \| 'bottom' \| 'left' \| 'right'` (combinable) | - | Which panels push content aside vs overlay |
| `children` | `ReactNode` | - | DrawerLayoutMain and panel components |

### DrawerLayoutMain

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Custom CSS class |
| `children` | `ReactNode` | - | Main content |

## How `extend` Works

```
extend: 'left right' (default)        extend: 'top bottom'
┌──────┬──────────┬───────┐           ┌─────────────────────────┐
│      │   top    │       │           │          top            │
│      ├──────────┤       │           ├──────┬──────────┬───────┤
│ left │   main   │ right │           │ left │   main   │ right │
│      ├──────────┤       │           ├──────┴──────────┴───────┤
│      │  bottom  │       │           │         bottom          │
└──────┴──────────┴───────┘           └─────────────────────────┘
```

## How `push` Works

| Value | Behavior |
|-------|----------|
| `undefined` | Panels float over main content (overlay) |
| `'left'` | Left panel pushes content to the right |
| `'left right'` | Both side panels push content |
| `'top bottom left right'` | All panels push content from all sides |

## Implementation Notes
- Built with CSS Grid for layout management
- Layout responds automatically to panel open/close states
- Must include exactly one `DrawerLayoutMain` component
