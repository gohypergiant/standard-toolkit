# Clock
An **unstyled**, self-updating `<time>` element displaying the current time. Updates every second.

## Example Usage

### Basic Clock
```tsx
<Clock />
```
<Canvas of={ClockStories.Default} />

### Styled Clock
```tsx
<Clock className="fg-accent-primary-bold" />
```
<Canvas of={ClockStories.Styling} />

### Custom Format
```tsx
const formatter = new Intl.DateTimeFormat('en-US', {
  dateStyle: 'short',
  timeStyle: 'long',
  timeZone: 'UTC',
  hour12: false,
});

<Clock formatter={formatter} />
```
<Canvas of={ClockStories.CustomFormat} />

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `formatter` | `Intl.DateTimeFormat` | See below | Custom date/time formatter |
| `className` | `string` | - | Additional CSS classes |

**Default formatter:**
```tsx
new Intl.DateTimeFormat('en-US', {
  timeStyle: 'long',
  timeZone: 'UTC',
  hour12: false,
})
```

## Implementation Notes
- Renders a `<time>` element
- Updates every second using `setClockInterval` from `@accelint/temporal`
- Unstyled by default - add your own styles via `className`
