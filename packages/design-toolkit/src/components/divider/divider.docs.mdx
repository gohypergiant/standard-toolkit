# Divider
A simple visual separator between content sections. Supports both horizontal and vertical orientations. Renders as a semantic `<hr>` element.

## Example Usage

```tsx
<Divider />
```
<Canvas of={DividerStories.Default} />

### Horizontal
```tsx
<div>
  <p>Content above</p>
  <Divider />
  <p>Content below</p>
</div>
```
<Canvas of={DividerStories.Horizontal} />

### Vertical
```tsx
<div className="flex gap-s">
  <span>Left</span>
  <Divider orientation="vertical" />
  <span>Right</span>
</div>
```
<Canvas of={DividerStories.Vertical} />

## Props

### Divider
Extends all standard HTML `<hr>` attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Direction of the divider |
| `className` | `string` | - | Custom CSS class |

### DividerProvider
Context provider for setting default props across multiple Divider components.

```tsx
<DividerProvider orientation="vertical">
  <Divider />
  <Divider />
</DividerProvider>
```

## Accessibility
- Uses semantic `<hr>` element
- Provides visual separation that assistive technologies can recognize

## Implementation Notes
- Automatically detects flexbox parent and adjusts sizing accordingly
- Horizontal dividers span full width; vertical dividers span full height
