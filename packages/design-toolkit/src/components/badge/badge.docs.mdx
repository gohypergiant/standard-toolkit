# Badge
The Badge component displays small status indicators, notifications, or labels that can be positioned relative to other elements. It's built with Tailwind CSS and provides excellent visual feedback for different states and priorities. Badges are commonly used to show notification counts, status indicators, priority levels, and small pieces of information that need to stand out from the surrounding content.

## Example Usage

### Basic
```tsx
<Badge>New</Badge>
```
<Canvas of={BadgeStories.Default} />

### With Text
```tsx
<Badge color="critical">99+</Badge>
```
<Canvas of={BadgeStories.WithText} />

### Dot Indicator (No Text)
```tsx
<Badge color="normal" />
```

### Positioned Badge
When composing with other components (like [Avatar](?path=/docs/components-avatar--playground)):
```tsx
import { designTokens } from '@accelint/design-foundation/tokens';

<Badge placement="top right" offset={designTokens.spacing.xs}>3</Badge>
```

### BadgeProvider
```tsx
<BadgeProvider color="info">
  <Badge>Status 1</Badge>
  <Badge>Status 2</Badge>
</BadgeProvider>
```

## Colors

| Color | Description |
|-------|-------------|
| `info` | Blue - informational content (default) |
| `advisory` | Yellow - advisory/warning content |
| `normal` | Green - normal/success states |
| `serious` | Orange - serious issues |
| `critical` | Red - critical errors or alerts |

## Props
Extends all standard HTML `span` attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `'info' \| 'advisory' \| 'normal' \| 'serious' \| 'critical'` | `'info'` | Visual color variant |
| `children` | `boolean \| null \| number \| string` | - | Badge content (empty for dot indicator) |
| `placement` | `'top' \| 'bottom' \| 'left' \| 'right' \| 'top left' \| 'top right' \| 'bottom left' \| 'bottom right'` | - | Position relative to parent |
| `offset` | `number \| { x?: number; y?: number }` | - | Distance from parent in pixels |
| `className` | `string` | - | Additional CSS classes |

### BadgeProvider
Context provider for setting default props across multiple badges.

```tsx
<BadgeProvider color="critical" placement="top right">
  <Badge>1</Badge>
  <Badge>2</Badge>
</BadgeProvider>
```

## Implementation Notes
- Renders as a `span` element
- Empty badges (no children) render as small dot indicators
- Badges with content expand with appropriate padding
- Uses CSS custom properties for positioning calculations
