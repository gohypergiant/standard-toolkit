# Tooltip

Contextual popup providing additional information on hover or focus.

```tsx
// Basic tooltip
<TooltipTrigger>
  <Button>Hover me</Button>
  <Tooltip>Helpful information</Tooltip>
</TooltipTrigger>

// With custom placement
<TooltipTrigger>
  <Button>Click me</Button>
  <Tooltip placement="top" offset={10}>
    Tooltip above the button
  </Tooltip>
</TooltipTrigger>

// With icon
<TooltipTrigger>
  <Icon><Info /></Icon>
  <Tooltip>Additional information</Tooltip>
</TooltipTrigger>

// With delay
<TooltipTrigger delay={500}>
  <Button>Delayed tooltip</Button>
  <Tooltip>Appears after 500ms</Tooltip>
</TooltipTrigger>
```

## TooltipTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Trigger element and Tooltip |
| `delay` | `number` | `250` | Delay before showing tooltip (ms) |
| `isDisabled` | `boolean` | — | Whether tooltip is disabled |

Extends React Aria [TooltipTrigger](https://react-spectrum.adobe.com/react-aria/Tooltip.html#tooltiptrigger) props.

## Tooltip Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Tooltip content |
| `placement` | `Placement` | `'bottom'` | Position relative to trigger |
| `offset` | `number` | `5` | Distance from trigger (px) |
| `parentRef` | `RefObject<HTMLElement>` | — | Portal container reference |
| `className` | `string` | — | Custom class for tooltip |

Extends React Aria [Tooltip](https://react-spectrum.adobe.com/react-aria/Tooltip.html#tooltip-1) props.

## Accessibility

- ARIA attributes for screen readers
- Keyboard navigation (Tab to focus, Escape to dismiss)
- Focus management
- Screen reader announcements
- Automatic viewport positioning
