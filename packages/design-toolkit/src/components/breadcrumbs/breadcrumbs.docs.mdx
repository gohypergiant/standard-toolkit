# Breadcrumbs
Accessible breadcrumb navigation built on React Aria Components. Displays the user's location within an application hierarchy with links back to parent pages. The final item represents the current page.

Components:
- **Breadcrumbs** - Container that renders the list of breadcrumb items
- **BreadcrumbItem** - Individual item; renders a link when `linkProps` is provided, otherwise renders as the current page

## Example Usage

```tsx
<Breadcrumbs>
  <BreadcrumbItem linkProps={{ href: '/' }}>Home</BreadcrumbItem>
  <BreadcrumbItem linkProps={{ href: '/projects' }}>Projects</BreadcrumbItem>
  <BreadcrumbItem>Project Phoenix</BreadcrumbItem>
</Breadcrumbs>
```

## Props

### Breadcrumbs
Extends all [`Breadcrumbs` props](https://react-spectrum.adobe.com/react-aria/Breadcrumbs.html#props) from react-aria-components.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | One or more `BreadcrumbItem` elements |
| `className` | `string` | - | Additional CSS classes |

### BreadcrumbItem
Extends [`Breadcrumb` props](https://react-spectrum.adobe.com/react-aria/Breadcrumbs.html#breadcrumb) from react-aria-components.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `linkProps` | `LinkProps` | - | Props for the link (include `href` to make clickable) |
| `children` | `ReactNode` | - | Item content |
| `classNames` | `object` | - | Custom class names (see below) |

**classNames:**
| Key | Description |
|-----|-------------|
| `item` | Applied to the breadcrumb item wrapper |
| `link` | Applied to the link element (when `linkProps` provided) |
| `separator` | Applied to the separator icon |

## Accessibility
- Keyboard navigation: Tab to focus, Enter to activate links
- ARIA landmark: Renders as `<nav>` with appropriate labeling
- Current page: Final item is marked as current via `aria-current="page"`

## Implementation Notes
- Wraps `Breadcrumbs` and `Breadcrumb` from react-aria-components
- Separator uses `ChevronRight` icon, hidden on final item
- Items without `linkProps` render as plain text (current page)
