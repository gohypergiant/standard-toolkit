# ActionBar
A container component that displays a collection of actions, typically represented as buttons with icons. The ActionBar provides consistent spacing and alignment for action items, commonly used in toolbars, editors, or command interfaces.

## Example Usage

```tsx
<ActionBar>
  <Button>
    <Icon><Copy /></Icon>
  </Button>
  <Button>
    <Icon><Delete /></Icon>
  </Button>
</ActionBar>
```

### With ToggleButton
ActionBar also supports [ToggleButton](?path=/story/components-button--toggle) for stateful actions.

```tsx
<ActionBar>
  <ToggleButton>
    <Icon><Bold /></Icon>
  </ToggleButton>
  <ToggleButton>
    <Icon><Italic /></Icon>
  </ToggleButton>
</ActionBar>
```

## Props
Extends all standard HTML `div` attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `elevation` | `'flat' \| 'overlay' \| 'raised'` | `'overlay'` | Shadow/elevation style |
| `size` | `'medium' \| 'small'` | `'medium'` | Size variant applied to child buttons |
| `children` | `ReactNode` | - | Button or ToggleButton elements |
| `className` | `string` | - | Additional CSS classes |

## Elevation
Controls the visual prominence of the ActionBar.

| Value | Use Case |
|-------|----------|
| `flat` | Embedded in panels with existing visual boundaries |
| `overlay` | Floating over content (default) |
| `raised` | Higher prominence, stronger shadow |

```tsx
<ActionBar elevation="flat">
  <Button>
    <Icon><Copy /></Icon>
  </Button>
</ActionBar>
```

## Implementation Notes
- Renders as a `<nav>` element for semantic navigation
- Automatically provides `variant="icon"` context to child Button and ToggleButton components
- Size prop propagates to all child buttons via context
