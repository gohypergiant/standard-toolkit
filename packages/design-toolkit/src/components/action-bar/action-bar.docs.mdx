# ActionBar
A container component that displays a collection of actions, typically represented as buttons with icons. The ActionBar provides consistent spacing and alignment for action items, commonly used in toolbars, editors, or command interfaces.


## Example usage
```
<ActionBar>
  <Button>
    <Icon>
      <Copy />
    </Icon>
  </Button>
  <Button>
    <Icon>
      <Delete />
    </Icon>
  </Button>
</ActionBar>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `showElevation` | `boolean` | `true` | Controls visibility of the elevation shadow |
| `size` | `'medium' \| 'small'` | `'medium'` | Size variant for the action bar and its buttons |

## Without elevation
Use `showElevation={false}` when the ActionBar is embedded in a context that already has its own visual boundaries, such as entity detail panels.

```tsx
<ActionBar showElevation={false}>
  <Button>
    <Icon>
      <Copy />
    </Icon>
  </Button>
</ActionBar>
```
