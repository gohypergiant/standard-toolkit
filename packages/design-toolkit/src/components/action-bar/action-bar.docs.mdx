# ActionBar
A container component that displays a collection of actions, typically represented as buttons with icons. The ActionBar provides consistent spacing and alignment for action items, commonly used in toolbars, editors, or command interfaces.


## Example usage
```
<ActionBar>
  <Button>
    <Icon>
      <Copy />
    </Icon>
  </Button>
  <Button>
    <Icon>
      <Delete />
    </Icon>
  </Button>
</ActionBar>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `elevation` | `string` | `overlay` | Elevation shadow style: `default`, `overlay`, or `raised` |
| `size` | `string` | `medium` | Size variant: `medium` or `small` |

## Elevation options
Use `elevation="default"` when the ActionBar is embedded in a context that already has its own visual boundaries, such as entity detail panels.

```tsx
<ActionBar elevation="default">
  <Button>
    <Icon>
      <Copy />
    </Icon>
  </Button>
</ActionBar>
```
