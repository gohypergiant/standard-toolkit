# Carousel

A carousel component for displaying a collection of items.

Its layout is customizable using the following components:
- **CarouselViewer** - Main section that displays the current item selected.
- **CarouselPrevious** - Move to previous item, disabled when first item is active.
- **CarouselNext** - Move to next item, disabled when last item is selected.
- **CarouselGallery** - A gallery that displays selectable thumbnails for items.
- **CarouselSelect** - A select field that displays a list of items in a **`SelectField`** component.
- **CarouselDisplay** - Displays current position and total count of items.

## Example Usage

```tsx
const [currentPosition, setCurrentPosition] = useState(0);
<Carousel
  variant="gallery"
  items={[
    {
      dataType: 'image',
      dataUrl: '[https://example.com/image1.jpg'](https://example.com/image1.jpg'),
      fileName: 'image1.jpg',
      title: 'Image 1',
      thumbnailUrl: '[https://example.com/thumbnail1.jpg'](https://example.com/thumbnail1.jpg'),
      uuid: '1',
    },
    {
      dataType: 'image',
      dataUrl: '[https://example.com/image2.jpg'](https://example.com/image2.jpg'),
      fileName: 'image2.jpg',
      title: 'Image 2',
      thumbnailUrl: '[https://example.com/thumbnail2.jpg'](https://example.com/thumbnail2.jpg'),
      uuid: '2',
    },
    // ...
  ]}
  currentPosition={currentPosition}
  setCurrentPosition={setCurrentPosition}
>
  <div className="flex flex-column">
    <CarouselViewer />
    <div className="flex flex-row justify-between gap-s">
      <CarouselPrevious />
      <CarouselGallery />
      <CarouselNext />
    </div>
  </div>
</Carousel>
```

## Props

### Carousel
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `Array<CarouselData>` | - | The items to display in the carousel. |

#### CarouselData
The data type for items in the carousel.

| Field | Type | Description |
|-------|------|-------------|
| `dataType` | `string` | The data type of the item. |
| `dataUrl` | `string` | The data URL of the item. |
| `fileName` | `string` | The file name of the item. |
| `metadata` | `object` | The metadata of the item. |
| `title` | `string` | The title of the item. |
| `thumbnailUrl` | `string` | The thumbnail URL of the item. |
| `uuid` | `string` | The unique identifier of the item. |


### CarouselViewer
Main image display of the carousel.

|Prop|Type|Default|Description|
|----|----|-------|-----------|
|`children`|`ReactNode`|-|Additional content to display in the carousel viewer. By default displays an image.|
|`className`|`string`|-|The class name for the carousel viewer.|

### CarouselGallery
|Prop|Type|Default|Description|
|----|----|-------|-----------|
|`classNames`|`object`|-|The class names for the carousel gallery.|

#### **`classNames`**
- **`container`** - The class name for the carousel gallery container.
- **`item`** - The class name for the carousel gallery item.

### CarouselPrevious
Navigates to the previous item in the carousel.

|Prop|Type|Default|Description|
|----|----|-------|-----------|
|`className`|`string`|-|The class name for the carousel previous button.|

### CarouselNext
Navigates to the next item in the carousel.

|Prop|Type|Default|Description|
|----|----|-------|-----------|
|`className`|`string`|-|The class name for the carousel next button.|

### CarouselPosition
Displays current position and total count of items in the carousel.

|Prop|Type|Default|Description|
|----|----|-------|-----------|
|`className`|`string`|-|The class name for the carousel position.|

### CarouselSelect
**`SelectField`** component that displays a list of items in the carousel.

|Prop|Type|Default|Description|
|----|----|-------|-----------|
|`classNames`|`object`|-|The class names for the carousel select.|

#### **`classNames`**
- **`field`** - The class name for the carousel select field.
- **`optionItem`** - The class name for the carousel select option item.


## Implementation Notes
- Currently limited to displaying images.
