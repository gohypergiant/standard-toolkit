# Skeleton

Placeholder content for loading states with a pulsing animation.

```tsx
<Skeleton />

<Skeleton variant="circle" />

<Skeleton>Loading content...</Skeleton>

<Skeleton className="mb-xl p-m">
  <Skeleton className="w-[75%]" />
</Skeleton>
```

<Canvas of={SkeletonStories.Default} />

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'rectangle' \| 'circle'` | `'rectangle'` | Shape of the placeholder |
| `className` | `string` | — | Custom CSS class |
| `children` | `ReactNode` | — | Nested content or placeholders |

Accepts all standard HTML div attributes.

## Accessibility

- Respects `prefers-reduced-motion` for animations
- Semantic HTML structure
