# ClassificationBadge
Displays security classification levels for documents, data, and information systems. Provides standardized visual indicators for different classification levels commonly used in government and secure environments.

## Example Usage

```tsx
<ClassificationBadge variant="unclassified" />
<ClassificationBadge variant="cui" />
<ClassificationBadge variant="confidential" />
<ClassificationBadge variant="secret" />
<ClassificationBadge variant="top-secret" />
<ClassificationBadge variant="ts-sci" />

// With custom text
<ClassificationBadge variant="confidential">CONFIDENTIAL//NOFORN</ClassificationBadge>

// Different sizes
<ClassificationBadge variant="secret" size="medium" />
<ClassificationBadge variant="secret" size="small" />
```

## Variants

| Variant | Default Text | Description |
|---------|--------------|-------------|
| `missing` | Missing | Red styling for missing classification (default) |
| `unclassified` | Unclassified | Gray styling for unclassified information |
| `cui` | CUI | Controlled Unclassified Information |
| `confidential` | Confidential | Confidential level information |
| `secret` | Secret | Secret level information |
| `top-secret` | Top Secret | Top secret level information |
| `ts-sci` | TS/SCI | Top Secret/Sensitive Compartmented Information |

## Sizes

| Size | Description |
|------|-------------|
| `medium` | Standard size (default) |
| `small` | Compact size for space-constrained layouts |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'missing' \| 'unclassified' \| 'cui' \| 'confidential' \| 'secret' \| 'top-secret' \| 'ts-sci'` | `'missing'` | Classification level |
| `size` | `'medium' \| 'small'` | `'medium'` | Size variant |
| `children` | `ReactNode` | - | Custom text (uses variant default text if empty) |
| `className` | `string` | - | Additional CSS classes |

### ClassificationBadgeProvider
Context provider for setting default props across multiple badges.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'missing' \| 'unclassified' \| 'cui' \| 'confidential' \| 'secret' \| 'top-secret' \| 'ts-sci'` | - | Classification level for all child badges |
| `size` | `'medium' \| 'small'` | - | Size for all child badges |
| `children` | `ReactNode` | - | ClassificationBadge components |

```tsx
<ClassificationBadgeProvider variant="secret" size="small">
  <ClassificationBadge />
  <ClassificationBadge />
</ClassificationBadgeProvider>
```

## Accessibility
- Proper contrast ratios for all classification levels
- Semantic color usage following security standards
- Renders as `<span>` element

## Implementation Notes
- Renders a `<span>` element
- Automatically displays default text for each variant when no children provided
- Color scheme follows standard security classification conventions
