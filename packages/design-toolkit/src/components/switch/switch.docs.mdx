# Switch

Toggle control for binary on/off states.

```tsx
<Switch>Enable notifications</Switch>

<Switch isSelected={isEnabled} onChange={setIsEnabled}>
  Dark mode
</Switch>

<Switch isDisabled>Feature unavailable</Switch>

<SwitchProvider labelPosition="start">
  <Switch>Option 1</Switch>
  <Switch>Option 2</Switch>
</SwitchProvider>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Label text |
| `isSelected` | `boolean` | — | Controlled selected state |
| `defaultSelected` | `boolean` | — | Initial selected state |
| `onChange` | `(isSelected: boolean) => void` | — | Callback when selection changes |
| `isDisabled` | `boolean` | — | Whether the switch is disabled |
| `labelPosition` | `'start' \| 'end'` | `'end'` | Position of the label relative to the control |
| `autoFocus` | `boolean` | — | Automatically focus on mount |
| `classNames` | `object` | — | Custom classes for internal elements |

Extends React Aria [Switch](https://react-spectrum.adobe.com/react-aria/Switch.html) props.

### classNames

| Key | Description |
|-----|-------------|
| `switch` | Switch container |
| `control` | Toggle control element |
| `label` | Label text |

## SwitchProvider

Sets default props for all Switch components within. Any Switch prop can be passed and will apply to all children unless overridden.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Switch components |
| `labelPosition` | `'start' \| 'end'` | `'end'` | Default label position |
| `isDisabled` | `boolean` | — | Default disabled state |
| `classNames` | `object` | — | Default custom classes |

## Accessibility

- Full keyboard support (Space/Enter to toggle)
- ARIA attributes for screen readers
- Focus management and visible focus indicators
- State announcements for assistive technology
