# ClassificationBanner
Full-width banner for displaying security classification levels at the top or bottom of pages, documents, and applications. Designed for government and secure environments requiring prominent classification indicators.

## Example Usage

```tsx
<ClassificationBanner variant="unclassified" />
<ClassificationBanner variant="cui" />
<ClassificationBanner variant="confidential" />
<ClassificationBanner variant="secret" />
<ClassificationBanner variant="top-secret" />
<ClassificationBanner variant="ts-sci" />

// With custom text
<ClassificationBanner variant="secret">
  SECRET - AUTHORIZED PERSONNEL ONLY
</ClassificationBanner>

// Page layout example
<div className="min-h-screen flex flex-col">
  <ClassificationBanner variant="secret" />
  <main className="flex-1 p-4">
    <h1>Page Content</h1>
  </main>
  <ClassificationBanner variant="secret" />
</div>
```

## Variants

| Variant | Default Text | Description |
|---------|--------------|-------------|
| `missing` | Missing | Red styling for missing classification (default) |
| `unclassified` | Unclassified | Gray styling for unclassified information |
| `cui` | CUI | Controlled Unclassified Information |
| `confidential` | Confidential | Confidential level information |
| `secret` | Secret | Secret level information |
| `top-secret` | Top Secret | Top secret level information |
| `ts-sci` | Top Secret/SCI | Top Secret/Sensitive Compartmented Information |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'missing' \| 'unclassified' \| 'cui' \| 'confidential' \| 'secret' \| 'top-secret' \| 'ts-sci'` | `'missing'` | Classification level |
| `children` | `ReactNode` | - | Custom text (uses variant default text if empty) |
| `className` | `string` | - | Additional CSS classes |

### ClassificationBannerProvider
Context provider for setting default props across multiple banners.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'missing' \| 'unclassified' \| 'cui' \| 'confidential' \| 'secret' \| 'top-secret' \| 'ts-sci'` | - | Classification level for all child banners |
| `children` | `ReactNode` | - | ClassificationBanner components |

```tsx
<ClassificationBannerProvider variant="secret">
  <ClassificationBanner />
  <ClassificationBanner />
</ClassificationBannerProvider>
```

## Accessibility
- Proper contrast ratios for all classification levels
- Semantic color usage following security standards
- Non-selectable text to prevent accidental copying

## Implementation Notes
- Renders a full-width `<div>` element
- Automatically displays default text for each variant when no children provided
- Color scheme follows standard security classification conventions
- Designed for placement at top/bottom of pages
