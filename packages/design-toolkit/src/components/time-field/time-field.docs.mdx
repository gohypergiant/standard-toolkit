# TimeField

Form input for entering and editing time values with configurable granularity.

```tsx
import { parseTime } from '@internationalized/date';

// Basic time field
<TimeField label="Start Time" />

// With default value and granularity
<TimeField
  label="Meeting Time"
  defaultValue={parseTime('14:30')}
  granularity="minute"
  description="Select meeting start time"
/>

// With validation
<TimeField
  label="Deadline"
  errorMessage="Please enter a valid time"
  isInvalid
  isRequired
/>

// Small size
<TimeField label="Time" size="small" />

// Readonly
<TimeField
  label="Recorded Time"
  defaultValue={parseTime('09:15:30')}
  isReadOnly
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Label text |
| `description` | `string` | — | Helper text below the input |
| `errorMessage` | `string` | — | Error text when invalid |
| `size` | `'medium' \| 'small'` | `'medium'` | Size variant |
| `granularity` | `'hour' \| 'minute' \| 'second'` | `'second'` | Time precision level |
| `hourCycle` | `12 \| 24` | `24` | 12-hour or 24-hour format |
| `shouldForceLeadingZeros` | `boolean` | `true` | Force leading zeros on segments |
| `defaultValue` | `TimeValue` | — | Initial time value |
| `value` | `TimeValue` | — | Controlled time value |
| `onChange` | `(value: TimeValue) => void` | — | Change handler |
| `isDisabled` | `boolean` | — | Whether the field is disabled |
| `isReadOnly` | `boolean` | — | Whether the field is readonly |
| `isRequired` | `boolean` | — | Whether the field is required |
| `isInvalid` | `boolean` | — | Whether the field is invalid |
| `inputProps` | `DateInputProps` | — | Props passed to the date input |
| `classNames` | `object` | — | Custom classes for internal elements |

Extends React Aria [TimeField](https://react-spectrum.adobe.com/react-aria/TimeField.html) props.

### classNames

| Key | Description |
|-----|-------------|
| `field` | Field container |
| `label` | Label element |
| `control` | Control wrapper containing icon and input |
| `input` | Date input element |
| `segment` | Individual time segment |
| `description` | Description text |
| `error` | Error message |
