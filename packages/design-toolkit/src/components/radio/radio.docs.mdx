# Radio
Individual radio button option for use within a RadioGroup. Built on React Aria Components with accessibility out of the box.

Components:
- **Radio** - Individual radio button option
- **[RadioGroup](?path=/docs/components-radiogroup--playground)** - Container for grouping radio buttons

## Example Usage

```tsx
<Radio value="red">Red</Radio>
```
<Canvas of={RadioStories.Default} />

## Props
Extends all [`Radio` props](https://react-spectrum.adobe.com/react-aria/RadioGroup.html#radio-1) from react-aria-components.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Label text |
| `value` | `string` | — | Value associated with this option |
| `isDisabled` | `boolean` | `false` | Disables this specific option |
| `labelPosition` | `'start' \| 'end'` | `'end'` | Position of the label relative to the control |
| `classNames` | `object` | — | Custom class names (see below) |

**classNames:**
| Key | Description |
|-----|-------------|
| `radio` | Applied to the radio container |
| `control` | Applied to the radio control (circle) |
| `label` | Applied to the label text |

## Accessibility
- Arrow keys navigate between options
- Space/Enter selects the focused option
- Automatic ARIA attributes for screen readers
- Focus management and form integration

## Implementation Notes
- Wraps `Radio` from react-aria-components
- Must be used inside a RadioGroup
- Inherits `labelPosition` from RadioGroup when not set directly
