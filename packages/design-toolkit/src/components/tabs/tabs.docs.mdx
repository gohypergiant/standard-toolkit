# Tabs

Tab navigation for organizing content into multiple sections.

```tsx
// Basic horizontal tabs
<Tabs defaultSelectedKey="tab1">
  <TabList>
    <Tab id="tab1">Overview</Tab>
    <Tab id="tab2">Details</Tab>
    <Tab id="tab3">Settings</Tab>
  </TabList>
  <TabPanel id="tab1">Overview content</TabPanel>
  <TabPanel id="tab2">Details content</TabPanel>
  <TabPanel id="tab3">Settings content</TabPanel>
</Tabs>

// Vertical tabs
<Tabs orientation="vertical">
  <TabList>
    <Tab id="profile">Profile</Tab>
    <Tab id="account">Account</Tab>
  </TabList>
  <TabPanel id="profile">Profile settings</TabPanel>
  <TabPanel id="account">Account settings</TabPanel>
</Tabs>

// Icon tabs
<Tabs>
  <TabList>
    <Tab id="home"><Icon><Home /></Icon></Tab>
    <Tab id="search"><Icon><Search /></Icon></Tab>
  </TabList>
  <TabPanel id="home">Home content</TabPanel>
  <TabPanel id="search">Search content</TabPanel>
</Tabs>
```

## Props

### Tabs

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `'start' \| 'center' \| 'end'` | `'start'` | — | Determines content alignment within Tab |
| `defaultSelectedKey` | `Key` | — | Initial selected tab key |
| `flex` | `boolean` | true | Causes tab size to grow up to a max width |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Tab orientation |
| `selectedKey` | `Key` | — | Controlled selected tab key |
| `isDisabled` | `boolean` | — | Disable all tabs |
| `onSelectionChange` | `(key: Key) => void` | — | Callback when selection changes |
| `children` | `ReactNode` | — | TabList and TabPanel components |

Extends React Aria [Tabs](https://react-spectrum.adobe.com/react-aria/Tabs.html) props.

### TabList

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Tab components |
| `className` | `string` | — | Custom CSS class |

### Tab

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `id` | `Key` | — | Unique identifier matching a TabPanel |
| `align` | `'start' \| 'center' \| 'end'` | `'start'` | — | Determines content alignment within Tab |
| `flex` | `boolean` | true | Causes tab size to grow up to a max width |
| `isDisabled` | `boolean` | — | Whether the tab is disabled |
| `children` | `ReactNode` | — | Tab content (text or icon) |

### TabPanel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `id` | `Key` | — | Unique identifier matching a Tab |
| `children` | `ReactNode` | — | Panel content |

## TabsProvider

Sets default props for all Tabs components within. Any Tabs prop can be passed and will apply to all children unless overridden.

## Accessibility

- Arrow key navigation between tabs
- Proper ARIA roles and attributes
- Focus management for keyboard users
- Screen reader announcements for selection
