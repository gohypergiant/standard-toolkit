# TextAreaField

Multi-line text input with integrated label, description, and error handling.

```tsx
// Basic text area
<TextAreaField label="Description" />

// With validation
<TextAreaField
  label="Feedback"
  description="Please provide detailed feedback"
  errorMessage="Feedback is required"
  isInvalid
  isRequired
/>

// With custom textarea props
<TextAreaField
  label="Bio"
  inputProps={{
    placeholder: "Tell us about yourself...",
    rows: 4,
    maxLength: 500
  }}
/>

// Small size
<TextAreaField label="Notes" size="small" />

// Readonly
<TextAreaField
  label="Summary"
  isReadOnly
  inputProps={{ value: 'Readonly content' }}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Label text |
| `description` | `string` | — | Helper text below the input |
| `errorMessage` | `ReactNode` | — | Error text when invalid |
| `size` | `'medium' \| 'small'` | `'medium'` | Size variant |
| `isDisabled` | `boolean` | — | Whether the field is disabled |
| `isReadOnly` | `boolean` | — | Whether the field is readonly |
| `isRequired` | `boolean` | — | Whether the field is required |
| `isInvalid` | `boolean` | — | Whether the field is invalid |
| `inputProps` | `TextAreaProps` | — | Props passed to the textarea element |
| `classNames` | `object` | — | Custom classes for internal elements |

Extends React Aria [TextField](https://react-spectrum.adobe.com/react-aria/TextField.html) props.

### classNames

| Key | Description |
|-----|-------------|
| `field` | Field container |
| `label` | Label element |
| `input` | Textarea element |
| `description` | Description text |
| `error` | Error message |

## TextAreaFieldProvider

Sets default props for all TextAreaField components within. Any TextAreaField prop can be passed and will apply to all children unless overridden.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | TextAreaField components |
| `size` | `'medium' \| 'small'` | `'medium'` | Default size |
| `isDisabled` | `boolean` | — | Default disabled state |
| `classNames` | `object` | — | Default custom classes |
