# Notice

A notification component for displaying temporary messages, alerts, and user feedback.

## Basic

```tsx
<Notice
  id="7a3b6dfd-6596-472c-af1a-06d433958705"
  message="This is a basic notice message"
/>
```

<Canvas of={NoticeStories.Default} />

## With Actions

```tsx
<Notice
  id="39f174fc-ead3-46f4-9131-239ac2a7ed27"
  color="serious"
  message="Are you sure you want to delete this item?"
  primary={{ children: "Delete" }}
  secondary={{ children: "Cancel" }}
  shouldCloseOnAction={true}
  onPrimaryAction={() => console.log('Primary action: Delete was pressed')}
  onSecondaryAction={() => console.log('Secondary action: Cancel was pressed')}
/>
```

## With Close Button

```tsx
<Notice
  id="7765f06c-dc2a-4041-b8c0-3f64f8cd2c34"
  color="critical"
  message="An error occurred while processing your request"
  showClose={true}
  onClose={() => console.log('Close was pressed')}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `id` | `string` | - | Unique identifier (required) |
| `message` | `string` | - | Text content to display (required) |
| `color` | `'info' \| 'advisory' \| 'normal' \| 'serious' \| 'critical'` | `'info'` | Visual style variant |
| `size` | `'medium' \| 'small'` | `'medium'` | Size variant |
| `primary` | `ButtonProps` | - | Primary action button config |
| `secondary` | `ButtonProps` | - | Secondary action button config |
| `hideIcon` | `boolean` | `false` | Whether to hide the icon |
| `showClose` | `boolean` | `false` | Whether to show close button |
| `shouldCloseOnAction` | `boolean` | `false` | Close on action press |
| `onPrimaryAction` | `() => void` | - | Primary button callback |
| `onSecondaryAction` | `() => void` | - | Secondary button callback |
| `onClose` | `() => void` | - | Close button callback |
| `classNames` | `{ notice?, content?, message?, actions? }` | - | Custom class names |

> **Note:** Icons are automatically displayed based on the color variant:
> - **info/advisory**: Information icon
> - **normal**: Success/checkmark icon
> - **serious**: Warning icon
> - **critical**: Problem/error icon
