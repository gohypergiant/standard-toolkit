# DetailsList
Semantic details list for displaying metadata in key-value pairs. Uses CSS Grid layout with labels in the first column and values in the second.

Components:
- **DetailsList** - Container with grid layout and alignment options
- **DetailsListLabel** - Key/term label (renders as `<dt>`)
- **DetailsListValue** - Value/definition (renders as `<dd>`)

## Example Usage

```tsx
<DetailsList align="justify">
  <DetailsListLabel>Key</DetailsListLabel>
  <DetailsListValue>Value</DetailsListValue>

  <DetailsListLabel>Ships</DetailsListLabel>
  <DetailsListValue>Millennium Falcon</DetailsListValue>
  <DetailsListValue>USS Enterprise NCC-1701</DetailsListValue>
  <DetailsListValue>Serenity</DetailsListValue>

  <DetailsListLabel>Coordinates</DetailsListLabel>
  <DetailsListValue>
    <div>Great Pyramid of Giza: 29째58'44" N 31째08'02" E</div>
    <div>Machu Picchu: 13째09'47" S 72째32'41" W</div>
  </DetailsListValue>
</DetailsList>
```

## Alignments

| Align | Description |
|-------|-------------|
| `left` | Both labels and values left-aligned |
| `center` | Labels right-aligned, values left-aligned |
| `justify` | Labels left-aligned, values right-aligned (default) |

## Props

### DetailsList

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `'left' \| 'center' \| 'justify'` | `'justify'` | Text alignment variant |
| `classNames` | `object` | - | Custom classes (see below) |

**classNames:**

| Key | Description |
|-----|-------------|
| `list` | Main container (`<dl>` element) |
| `label` | Label elements (`<dt>` elements) |
| `value` | Value elements (`<dd>` elements) |

### DetailsListLabel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Label content |
| `className` | `string` | - | Custom CSS class |

### DetailsListValue

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Value content (supports complex React nodes) |
| `className` | `string` | - | Custom CSS class |

### DetailsListProvider
Context provider for setting default props across multiple DetailsList components.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `'left' \| 'center' \| 'justify'` | - | Alignment for all child DetailsLists |
| `classNames` | `object` | - | Custom classes for all child DetailsLists |
| `children` | `ReactNode` | - | DetailsList components |

```tsx
<DetailsListProvider align="left">
  <DetailsList>...</DetailsList>
  <DetailsList>...</DetailsList>
</DetailsListProvider>
```

## Accessibility
- Uses semantic HTML: `<dl>`, `<dt>`, `<dd>` elements
- Proper structure for screen readers and assistive technologies

## Implementation Notes
- Renders a `<dl>` element with CSS Grid layout
- DetailsListLabel renders as `<dt>`
- DetailsListValue renders as `<dd>`
- Supports multiple values per label
