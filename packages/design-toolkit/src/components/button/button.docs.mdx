# Button
Interactive elements for user actions, navigation, and form submissions. Built on React Aria Components with accessibility out of the box.

Components:
- **Button** - Standard button for actions
- **LinkButton** - Button-styled link for navigation
- **ToggleButton** - Button with pressed/unpressed state

## Example Usage

### Button
```tsx
<Button>Click me</Button>
<Button variant="outline" color="mono-bold">Outline Button</Button>
<Button variant="flat">Flat Button</Button>
<Button variant="icon">
  <Icon><Settings /></Icon>
</Button>
```
<Canvas of={ButtonStories.Default} />

### LinkButton
Button-styled link for navigation.
```tsx
<LinkButton href="/dashboard" variant="outline">
  Go to Dashboard
</LinkButton>
```
<Canvas of={ButtonStories.Link} />

### ToggleButton
Button with pressed/unpressed state. Note: `filled` variant is not available for ToggleButton.
```tsx
<ToggleButton variant="flat">Toggle Feature</ToggleButton>
<ToggleButton variant="outline" color="accent">Toggle with Color</ToggleButton>
<ToggleButton variant="icon">
  <Icon><Placeholder /></Icon>
</ToggleButton>
```

## Variants

| Variant | Description | Available On |
|---------|-------------|--------------|
| `filled` | Solid background (default) | Button, LinkButton |
| `outline` | Transparent background with border | All |
| `flat` | Transparent background, text only | All |
| `icon` | Square button for icon-only content | All |

## Colors

| Color | Description |
|-------|-------------|
| `mono-muted` | Monochromatic styling (default) |
| `mono-bold` | Monochromatic with bolder accents |
| `accent` | Primary accent for main actions |
| `serious` | Orange for warning/caution actions |
| `critical` | Red for destructive actions |

## Sizes

| Size | Description |
|------|-------------|
| `large` | Prominent size for primary actions |
| `medium` | Standard size (default) |
| `small` | Compact size for secondary actions |
| `xsmall` | Minimal size for tight spaces |

## Props

### Button
Extends all [`Button` props](https://react-spectrum.adobe.com/react-aria/Button.html#props) from react-aria-components.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'filled' \| 'outline' \| 'flat' \| 'icon'` | `'filled'` | Visual style |
| `color` | `'mono-muted' \| 'mono-bold' \| 'accent' \| 'serious' \| 'critical'` | `'mono-muted'` | Color scheme |
| `size` | `'large' \| 'medium' \| 'small' \| 'xsmall'` | `'medium'` | Size variant |
| `children` | `ReactNode` | - | Button content |
| `className` | `string` | - | Additional CSS classes |
| `isDisabled` | `boolean` | `false` | Whether the button is disabled |
| `onPress` | `(e: PressEvent) => void` | - | Called when button is pressed |

### LinkButton
Extends all [`Link` props](https://react-spectrum.adobe.com/react-aria/Link.html#props) from react-aria-components.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | - | URL to navigate to |
| `variant` | `'filled' \| 'outline' \| 'flat' \| 'icon'` | `'filled'` | Visual style |
| `color` | `'mono-muted' \| 'mono-bold' \| 'accent' \| 'serious' \| 'critical'` | `'mono-muted'` | Color scheme |
| `size` | `'large' \| 'medium' \| 'small' \| 'xsmall'` | `'medium'` | Size variant |

### ToggleButton
Extends all [`ToggleButton` props](https://react-spectrum.adobe.com/react-aria/ToggleButton.html#props) from react-aria-components.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'outline' \| 'flat' \| 'icon'` | `'flat'` | Visual style (no `filled`) |
| `color` | `'mono-muted' \| 'mono-bold' \| 'accent' \| 'serious' \| 'critical'` | `'mono-muted'` | Color scheme |
| `size` | `'large' \| 'medium' \| 'small' \| 'xsmall'` | `'medium'` | Size variant |
| `isSelected` | `boolean` | - | Controlled selected state |
| `defaultSelected` | `boolean` | `false` | Default selected state |
| `onChange` | `(isSelected: boolean) => void` | - | Called when selection changes |

### ButtonProvider
Context provider for setting default props across multiple buttons.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'filled' \| 'outline' \| 'flat' \| 'icon'` | - | Visual style for all child Buttons |
| `color` | `'mono-muted' \| 'mono-bold' \| 'accent' \| 'serious' \| 'critical'` | - | Color for all child Buttons |
| `size` | `'large' \| 'medium' \| 'small' \| 'xsmall'` | - | Size for all child Buttons |
| `children` | `ReactNode` | - | Button components |

```tsx
<ButtonProvider size="small" variant="outline">
  <Button>Button 1</Button>
  <Button>Button 2</Button>
</ButtonProvider>
```

### LinkButtonProvider

| `size` | `'large' \| 'medium' \| 'small' \| 'xsmall'` | - | Size for all child LinkButtons |

### ToggleButtonProvider

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'outline' \| 'flat' \| 'icon'` | - | Visual style for all child ToggleButtons |
| `color` | `'mono-muted' \| 'mono-bold' \| 'accent' \| 'serious' \| 'critical'` | - | Color for all child ToggleButtons |

## Accessibility
- Keyboard: Enter and Space to activate
- ARIA attributes for screen readers
- Focus management and visual focus indicators
- ToggleButton uses `aria-pressed` for state

## Implementation Notes
- Button wraps `Button` from react-aria-components
- LinkButton wraps `Link` from react-aria-components
- ToggleButton wraps `ToggleButton` from react-aria-components
- Icons automatically scale based on size variant via IconProvider

Underlying elements:
- `<button>` for Button and ToggleButton
- `<a>` for LinkButton
