# Checkbox
Interactive checkbox for form inputs, settings, and multi-selection interfaces. Built on React Aria Components with accessibility out of the box.

Components:
- **Checkbox** - Individual checkbox for binary selections
- **[CheckboxGroup](?path=/docs/components-checkboxgroup--playground)** - Container for grouping multiple checkboxes

## Example Usage

```tsx
<Checkbox>Accept terms and conditions</Checkbox>
```
<Canvas of={CheckboxStories.Default} />

### Controlled
```tsx
const [isSelected, setIsSelected] = useState(false);

<Checkbox isSelected={isSelected} onChange={setIsSelected}>
  Subscribe to newsletter
</Checkbox>
```

### Indeterminate
Used for "select all" scenarios with partial selection.
```tsx
<Checkbox isIndeterminate>Select all items</Checkbox>
```

## States

| State | Description |
|-------|-------------|
| Unselected | Default state, empty checkbox |
| Selected | Checked state with checkmark icon |
| Indeterminate | Partial selection with minus icon |
| Disabled | Non-interactive with reduced opacity |

## Props
Extends all [`Checkbox` props](https://react-spectrum.adobe.com/react-aria/Checkbox.html#props) from react-aria-components.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Label text |
| `isSelected` | `boolean` | - | Controlled selected state |
| `defaultSelected` | `boolean` | `false` | Default selected state |
| `isIndeterminate` | `boolean` | `false` | Partial selection state |
| `isDisabled` | `boolean` | `false` | Whether checkbox is disabled |
| `value` | `string` | - | Value when used in CheckboxGroup |
| `onChange` | `(isSelected: boolean) => void` | - | Called when selection changes |
| `classNames` | `object` | - | Custom class names (see below) |

**classNames:**
| Key | Description |
|-----|-------------|
| `checkbox` | Applied to the checkbox container |
| `control` | Applied to the checkbox control (square box) |
| `label` | Applied to the label text |

## Accessibility
- Keyboard: Space to toggle
- ARIA attributes for screen readers
- Focus management and visual indicators
- Supports form validation

## Implementation Notes
- Wraps `Checkbox` from react-aria-components
- Shows Check icon when selected, Remove icon when indeterminate
- Use within CheckboxGroup for multi-selection with shared state
