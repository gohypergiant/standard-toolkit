# TextField

Single-line text input with integrated label, description, and error handling.

```tsx
// Basic text field
<TextField label="Username" />

// With validation
<TextField
  label="Email"
  description="We'll never share your email"
  errorMessage="Please enter a valid email"
  isInvalid
  isRequired
  inputProps={{ type: "email", placeholder: "example@email.com" }}
/>

// With prefix and suffix
<TextField
  label="Price"
  inputProps={{ type: "number", prefix: "$", suffix: "USD" }}
/>

// Small size with clearable input
<TextField
  label="Search"
  size="small"
  inputProps={{ type: "search", isClearable: true }}
/>

// Readonly
<TextField
  label="Username"
  isReadOnly
  inputProps={{ value: "readonly value" }}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Label text |
| `description` | `string` | — | Helper text below the input |
| `errorMessage` | `ReactNode` | — | Error text when invalid |
| `size` | `'medium' \| 'small'` | `'medium'` | Size variant |
| `isDisabled` | `boolean` | — | Whether the field is disabled |
| `isReadOnly` | `boolean` | — | Whether the field is readonly |
| `isRequired` | `boolean` | — | Whether the field is required |
| `isInvalid` | `boolean` | — | Whether the field is invalid |
| `inputProps` | `InputProps` | — | Props passed to the input element |
| `classNames` | `object` | — | Custom classes for internal elements |

Extends React Aria [TextField](https://react-spectrum.adobe.com/react-aria/TextField.html) props.

### classNames

| Key | Description |
|-----|-------------|
| `field` | Field container |
| `label` | Label element |
| `input` | Input element (accepts Input classNames object) |
| `description` | Description text |
| `error` | Error message |

## TextFieldProvider

Sets default props for all TextField components within. Any TextField prop can be passed and will apply to all children unless overridden.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | TextField components |
| `size` | `'medium' \| 'small'` | `'medium'` | Default size |
| `isDisabled` | `boolean` | — | Default disabled state |
| `classNames` | `object` | — | Default custom classes |
