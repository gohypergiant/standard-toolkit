# Pagination
The Pagination component provides a simple way to navigate multiple pages while leveraging existing Button components.

It is composed of the following components:

- **Pagination** - A wrapper component for containing all pagination buttons. Logic is contained within this component.


## Example usage
```tsx
  <Pagination value={1} total={10} onChange={handleOnChange}>
```

<Canvas of={PaginationStories.Default} />

## Props

### Pagination Props
* **`value`** - Current page number to display, not zero-index.
* **`total`** - The total number of pages.
* **`isLoading`** - Current loading state.
* **`onChange`** - Handler function for the onPress event of control buttons.
* **`classNames`** - Custom class names for styling container
  * `container` - Control wrapper container.
  * `controls` - Previous/Next button styles.
  * `pages` - ToggleButtons corresponding to page numbers. Defaults to 5.
