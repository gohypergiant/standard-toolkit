# Pagination

A lightweight component for navigating between pages. Displays up to 5 page numbers at a time with previous/next navigation controls.

## Basic

```tsx
<Pagination value={1} total={10} onChange={handleChange} />
```

<Canvas of={PaginationStories.Default} />

## Controlled

```tsx
const [page, setPage] = useState(1);

<Pagination value={page} total={10} onChange={setPage} />
```

## Uncontrolled

```tsx
<Pagination defaultValue={1} total={10} onChange={handleChange} />
```

## Props

### Pagination

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | - | Current page (controlled, 1-indexed) |
| `defaultValue` | `number` | `1` | Initial page (uncontrolled) |
| `total` | `number` | `0` | Total number of pages |
| `isLoading` | `boolean` | `false` | Hides page numbers when loading |
| `onChange` | `(page: number) => void` | - | Called when page changes |
| `classNames` | `object` | - | Custom class names (see below) |

### classNames

| Key | Description |
|-----|-------------|
| `container` | The nav container |
| `prev` | Previous button |
| `pages` | Page number buttons |
| `next` | Next button |

## Accessibility

- Uses semantic `<nav>` element with `aria-label` showing current page
- Previous/next buttons have `aria-label` descriptions
- Current page has `aria-current="page"` attribute
