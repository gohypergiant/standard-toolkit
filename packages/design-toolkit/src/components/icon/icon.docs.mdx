# Icon
A wrapper component for displaying SVG icons with consistent sizing. Renders as a `<span>` element containing the icon SVG.

Components:
- **Icon** - Wrapper for SVG icons with size variants
- **IconProvider** - Context provider for default icon sizes

## Example Usage

```tsx
<Icon>
  <Add />
</Icon>

<Icon size="large">
  <Settings />
</Icon>
```

## Sizes

| Size | Pixels | Description |
|------|--------|-------------|
| `xsmall` | 16px | Dense interfaces and small UI elements |
| `small` | 20px | Secondary actions and compact layouts |
| `medium` | 24px | Standard size (default) |
| `large` | 32px | Prominent actions and main navigation |

## Props

### Icon
Extends all standard HTML `<span>` attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'xsmall' \| 'small' \| 'medium' \| 'large'` | `'medium'` | Icon size variant |
| `className` | `string` | - | Custom CSS class |
| `children` | `ReactNode` | - | SVG icon component |

### IconProvider
Context provider for setting default sizes across multiple icons.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'xsmall' \| 'small' \| 'medium' \| 'large'` | - | Size applied to all child Icons |
| `children` | `ReactNode` | - | Icon components |

```tsx
<IconProvider size="small">
  <Icon><Home /></Icon>
  <Icon><Search /></Icon>
</IconProvider>
```

## Accessibility
- Icons inherit color from parent context
- Use `aria-label` for standalone icons that convey meaning

```tsx
<Icon aria-label="Add item">
  <Add />
</Icon>
```

## Implementation Notes
- Uses CSS custom properties for sizing (`--icon-size-xs`, `--icon-size-s`, etc.)
- Automatically inherits size from IconProvider or parent component context
