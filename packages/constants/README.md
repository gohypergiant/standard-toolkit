# @accelint/constants

A collection of commonly used constant values for Accelint projects.

## Installation

```sh
npm install @accelint/constants
```

## Constants

### Coordinates

- **DEFAULT_COORDINATE** - Default empty coordinate value `[NaN, NaN]`. Using `[0, 0]` as a default is incorrect as it represents a real location in the Atlantic Ocean known as "Null Island".
