# @accelint/core

A set of utility functions for manipulating arrays, objects, logic, and function composition in JavaScript. This lightweight toolkit simplifies data manipulation and functional programming.

## Installation

```sh
npm install @accelint/core
```

## Categories

This package includes utilities across the following categories:

- **Array** - Array manipulation and query utilities
- **Combinators** - Lambda calculus combinators (identity, constant, fork, etc.)
- **Composition** - Function composition and piping utilities
- **Iterable** - Iterator and iterable creation utilities
- **Logical** - Logical operators (and, or, xor, not, etc.)
- **Object** - Object manipulation utilities (associate, lens, property)
- **Safe Enum** - Type-safe enum alternative to TypeScript enums
- **Utility** - General utilities (lookup, noop, once, tap, uuid)
- **Types** - Common type aliases for functional programming
