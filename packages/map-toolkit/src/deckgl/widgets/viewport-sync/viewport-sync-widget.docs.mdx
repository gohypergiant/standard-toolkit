# ViewportSyncWidget

A widget for the DeckGL that emits Viewport events from the Bus. Used by [ViewportSize](?path=/docs/viewport-viewport-size--docs) and [useViewportState](?path=/docs/viewport-useviewportstate--docs)

## Usage

```typescript
import { ViewportSyncWidget } from '@accelint/map-toolkit/deckgl/widgets/viewport-sync';
import { BaseMap } from '@accelint/map-toolkit/deckgl/base-map';
import { ViewportSize } from '@accelint/map-toolkit/viewport'

const VIEW_ID = 'default'
const WIDGETS = [new ViewportSyncWidget({ id: VIEW_ID })];

const Map = () => {
  return (
    <div>
      <ViewportSize viewId={VIEW_ID} />
      <BaseMap widgets={WIDGETS} />
    </div>
  )
}
```
