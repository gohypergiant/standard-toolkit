# Function: formatM3A()

> **formatM3A**(`value`?): `string`

A 4-digit octal code that identifies the aircraft. This mode is used by both the
military and civilians, and is often called a squawk code. Air traffic controllers
assign these codes.

## Parameters

• **value?**: `string` \| `number`

## Returns

`string`
